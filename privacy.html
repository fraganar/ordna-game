Senast uppdaterad: 25 oktober 2025

Vilken data samlar vi in?

Vi samlar bara in det som behövs för att spelet ska fungera:

  • Spelarnamn: Det namn du väljer att använda i spelet
  • Email: För att identifiera ditt konto mellan enheter
  • Speldata: Dina poäng, utmaningar och vilka frågepaket du spelat
  • Autentiseringsdata: Token från Google (om du loggar in med Google)

Vi samlar INTE in: IP-adresser, platsdata, cookies för annonsering, eller annan personlig information.

Var lagras datan?

All speldata lagras i Google Firebase (servrar i EU). Firebase följer GDPR och har säkra servrar. Din data är skyddad med Firebase Authentication.

Hur används datan?

  • För att visa dina poäng och utmaningar
  • För att matcha utmaningar mellan spelare
  • För att synka data mellan dina enheter
  • För att förbättra spelet (vi tittar på statistik, men aldrig på individuell data)

Vi delar ALDRIG din data med tredje part. Vi säljer inte data. Vi använder inte data för annonsering.

Cookies

Vi använder inga traditionella cookies. Firebase Authentication använder säker session-lagring för att hålla dig inloggad. Denna data hanteras automatiskt av Firebase och följer säkerhetsstandard.

Dina rättigheter

Du har rätt att:

  • Se din data: Kontakta oss så får du en kopia
  • Radera ditt konto: Kontakta oss så raderar vi all din data från Firebase
  • Ändra ditt namn: Gör det direkt i spelets inställningar

Säkerhet

Vi använder Firebase Authentication för att skydda din data. Endast du (och admin för felsökning) kan se din speldata. All kommunikation sker över HTTPS (krypterad).

Barn

Spelet är lämpligt för alla åldrar. Vi samlar inte medvetet in data från barn under 13 år. Om du är förälder och vill att ditt barns data ska raderas, kontakta oss.

Kontakt

Har du frågor om din integritet eller vill radera ditt konto? Använd "Rapportera problem"-funktionen i spelets meny så får du svar direkt.

TL;DR: Vi sparar bara det vi måste (namn, poäng, utmaningar). Data lagras säkert i Firebase. Vi säljer aldrig data. Du kan radera ditt konto när som helst. 🔒
